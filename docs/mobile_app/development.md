# Development
This section outlines development information and project structure for WATERWATCH mobile.

## Components
The components folder contains elements that are used across the app. For example, card_component.dart refers to the component that subdivides measurement input fields.

## Screens
The screens folder contains all of the screens used in the app. Existing screen files can be found and modified here and new screens should be added here.

## Utils
The utils folder contains functions associated with the mobile app. If certain functionality of the mobile app needs to be modified the functions to do so can be found here. For example, if you would like to modify how a measurement is stored you can do so via upload_stored_measurements.dart.

## Theme
The theme colors used in the app can be updated in main.dart.
