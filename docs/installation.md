# Installation
This section outlines the steps to install WATERWATCH.

## Development

Requirements before install:
- python
- docker

Linux:
```bash
./setup.sh
```
After installation the project will run at [localhost](http://127.0.0.1/).


There are two standard users:

Admin
    - username: admin
    - password: admin

Researcher
    - username: researcher
    - password: researcher
